# E-Commerce Sales Performance Dashboard

## Project Overview

This project analyzes an e-commerce dataset to identify key business insights such as revenue trends, top-selling products, high-performing countries, and top customers.

The goal of this analysis is to help businesses understand sales performance and make data-driven decisions.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning & Transformation

## Key Business Questions Answered

* Which products generate the most sales?
* How does revenue change over time?
* Which countries contribute the most revenue?
* Which customers generate the highest sales?

## Dashboard Features

* Revenue Trend Over Time
* Top 10 Selling Products
* Revenue by Country
* Top Customers by Revenue
* Interactive Filters (Month, Country, CustomerID)

## Key Insights

* United Kingdom contributes the majority of total revenue.
* Sales peak during November and December, indicating strong seasonal demand.
* A small number of customers generate a large share of revenue.
* Gift and decorative products dominate the sales volume.
